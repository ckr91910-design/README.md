# MRTC Bible OS

> **Mission**
>
> AI가 말씀을 대신하는 것이 아니라,
> 말씀을 따라 살아가는 창조적 제자를 세우는 플랫폼

---

## 프로젝트 소개

MRTC Bible OS는 **미리토크(MRTC)** 프레임워크를 기반으로 한
성경 말씀 체험 및 훈련 플랫폼입니다.

MRTC는 다음의 네 단계를 중심으로 말씀을 연구하고 삶에 적용합니다.

1. **Miqra (미크라)** – 말씀을 있는 그대로 관찰
2. **Rishma (리쉬마)** – 말씀을 마음으로 묵상
3. **Tota (토타)** – 성경 전체와 연결하여 검증
4. **Cristo (크리스토)** – 그리스도와 연합하여 삶으로 실천
